1. dir

This command is used to list the files and directories within the current directory in windows.

2. ls

This command is used to list the files and directories within the current directory in MacOS/Linux machine.

2. mkdir [folder_name]

This command is used to create a new directory with the specified name in Unix-like operating systems (such as Linux and macOS) and Windows.

3. cd [folder_name]

This command is used to change the current working directory to the specified directory in Unix-like operating systems (such as Linux and macOS) and Windows.

4. dir /a:h

This command in the windows command propmpt is used to list only hidden files and directories in the current directory.

5. ls -a

This command is used in Unix-like operating systems, such as Linux and macOS to list only hidden files and directories in the current directory.

6. dir [folder_name]

This command is used in Windows command prompt or PowerShell to display the files and subdirectories within the specified folder.

7. ls [folder_name]

This command is primarily used in Unix-based systems like Linux and macOS to display the files and subdirectories within the specified folder.

8. echo [Entire Text] > [File-Name].txt

This command is used to create a new text file with the specified [File-Name].txt and write the [Entire Text] into it.

9. touch [File-Name].txt

This command is commonly used in Unix-like operating systems (such as Linux and macOS) to create a new empty file.

10. vi [File_Name].txt

The vi command is used to open the Vim text editor in Unix-like operating systems, including Linux and macOS. When you type vi [File_Name].txt, you're instructing the system to open the Vim editor and create/edit the file [File_Name].txt.

11. more [File_Name].txt

This command is used to display the contents of a text file page by page in the windows terminal.

12. cat [File-Name].txt

This command is used in Unix-based systems like Linux and macOS to display the contents of a text file.

13. rm -rf [File-Name].txt

This command is used to remove a file named [File-Name].txt forcefully and recursively, along with all its contents and subdirectories (if it's a directory).
